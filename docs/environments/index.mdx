# Environments

The Nevermined Point of Sales App is available in both [production](https://nevermined.app/) and 
[testing](https://testing.nevermined.app/) environments. Both environments are open and ready to 
use. The main difference is that the [production](https://nevermined.app/) environment enables users to 
publish and process payments with real money.

The testing environment is especially useful for AI builders to understand and experiment with the Point 
of Sales application.

:::info

In the [testing](https://testing.nevermined.app/) environment, all payments use simulated money - no real currency is involved.

:::

In addition to these environments, we have **Headless Deployments** where we don't provide a Nevermined 
App, **BUT** the underlying network is available. These deployments are specifically oriented toward 
**Agent2Agent** communication and orchestration.
